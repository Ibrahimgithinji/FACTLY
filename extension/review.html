Implementation Summary

  • Backend: New POST /api/verification/quick-check/ endpoint (lightweight, 60 req/hr)
  • Extension: Chrome v3 manifest with context menus, auto-detect on 30+ news domains
  • Popup: SVG score ring, evidence list, history, "Check This Page" button
  • Content script: Inline floating badge + highlights sensational content
  • Keyboard shortcut: Ctrl+Shift+F (Cmd+Shift+F on Mac)
  • History: Saves last 50 verifications in chrome.storage.local
